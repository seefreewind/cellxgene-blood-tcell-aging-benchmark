# Figure Style Guide

- Canvas: horizontal, journal-width multipanel figures, white background.
- Font: sans-serif; SVG text is kept editable (`svg.fonttype = none`).
- Minimum intended font size: approximately 6-7 pt for dense labels, with legends/captions carrying longer explanations.
- Semantic colors: young = blue, old = red, scVI = green, TranscriptFormer = purple, metadata-only = gray, train = teal, test = orange, risk = muted red.
- Figure 1 uses schematic modules and rounded boxes for workflow/risk-control logic.
- Figure 2 uses data-derived plots from current metadata, hosted embeddings, and benchmark result tables.
- No rainbow palettes, no official logos, no copied layouts, and no unsupported biological or clinical claims.
